Portfolio 3 is the analysis of Mobile Price Dataset.

The portfolio starts by exploring the dataset and checking for null values. Null values were found and replaced with the average of the column they were found in. After that, I dropped any missing values.

After that I found the correlations regarding price_range which included ram, px_height  px_width, battery_power. I then displayed all of the correlations. 

Once that was completed I trained data containing 80% of the data. After that I trained the linear regression models and evaluated them using MSE and r^2.

After that, I trained a KNN model and found the training and testing accuracy and then trained the hyper-parameter K. 

Finally I visualised these results.
