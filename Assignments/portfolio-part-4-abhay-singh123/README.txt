In portfolio 4, displays my analysis of a red wine data set. 

The portfolio starts by exploring the dataset where I checked if there are any null values. Then I conducted univariate and bivariate data analysis using scatter plots. 

Then I found the most correlated features with my target variable of "quality". Once I did that I split the data into training and testing at a ratio of 8:2. 

Then I trained the linear regression model and evaluated it by finding the MSE and r^2. Then I visualised the difference of my predictions against the real values. 

Finally, I trained a KNN model and evaluated the training accuracy and testing accuracy.
